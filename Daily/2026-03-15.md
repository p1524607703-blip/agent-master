```json
POST https://api.scrapeless.com/api/v1/scraper/request
Header: x-api-token: <key>
Body:
{
  "actor": "scraper.shopee",
  "input": {
    "action": "shopee.search",
    "url": "https://shopee.tw/api/v4/search/search_items?by=ctime&limit=20&newest=0&order=desc&page_type=shop&scenario=PAGE_OTHERS&version=2&shopid={shopid}"
  }
}
```

- `by=ctime` → 最新上架
- `by=sales` → 热销排行
- Scrapeless API Key 填写位置：节点「监控目标配置」第一行 `SCRAPELESS_API_KEY`

#### 2. 修复 n8n Windows 社区节点安装失败（spawn npm ENOENT）

- **Issue**：[n8n #22706](https://github.com/n8n-io/n8n/issues/22706)
- **根本原因**：Windows 上 npm 是 `npm.cmd`，`execFile` 不走 Shell 找不到
- **修复文件**：`C:\Users\15246\AppData\Local\nvm\v20.20.1\node_modules\n8n\dist\modules\community-packages\npm-utils.js` 第 43 行
- **修改内容**：

```js
// 修复前
await asyncExecFile('npm', args, cwd ? { cwd } : undefined)

// 修复后
await asyncExecFile('npm', args, cwd ? { cwd, shell: true } : { shell: true })
```

## 遇到的问题 & 解决过程

### 问题 1：scraper.shopee actor 返回 code 15001 "actor not found"

**原因**：Scrapeless 账户 credits = 0，无法调用任何 actor
**验证方式**：
```bash
curl https://api.scrapeless.com/api/v1/me -H "x-api-token: <key>"
# 返回 credits: "0"
```
**解决**：登录 [app.scrapeless.com](https://app.scrapeless.com) 领取/充值免费额度（2000次）

### 问题 2：工作流用了不存在的 action 名

**错误 action**（已废弃）：
- `shopee.shop_new_products`
- `shopee.shop_bestseller`

**正确 action**（已更新）：
- `shopee.search`（传 Shopee 搜索 API URL）
- `shopee.product`（传商品详情页 URL）
- `shopee.live`（直播数据）

### 问题 3：n8n API PUT 请求报 "must NOT have additional properties"

**原因**：PUT body 包含了 `tags`（格式不对）等多余字段
**解决**：只保留 `name / nodes / connections / settings / staticData`，去掉其他字段

### 问题 4：Scrapeless 文档 URL 多处 404

部分文档路径需要从首页导航才能找到：
- Shopee actor 文档：`/en/scraping-api/features/scrape-shopee/`
- Universal Scraping API：`/en/universal-scraping-api/quickstart/getting-started/`
