## Test A 输出（无 Skill）

输入：`帮我调研一下夜间睡眠耳塞`

```
## 夜间睡眠耳塞调研

### 一、主要类型

**1. 海绵/泡沫耳塞（Foam）**
- 代表品牌：3M、Moldex、Howard Leight
- 降噪值（NRR）：通常 29–33 dB
- 优点：便宜（一次性）、隔音效果最好
- 缺点：长时间佩戴可能有胀痛感

**2. 硅胶耳塞（Silicone）**
- 代表品牌：Mack's Pillow Soft、Earpeace
- NRR：约 22–28 dB
- 优点：可水洗重复使用，侧睡更舒适

**3. 蜡质耳塞（Wax）**
- 代表品牌：Ohropax、Cirrus
- NRR：约 22–27 dB

**4. 定制耳塞（Custom-molded）**
- 价格：专业定制 500–2000 元

**5. 电子/主动降噪睡眠耳塞**
- 代表：Bose Sleepbuds II、1More SleepBuds
- 价格：500–2000 元

### 三、热门产品推荐（按价格段）
低价位（10–50 元）：3M 1100、Mack's Pillow Soft
中价位（50–200 元）：Loop Quiet、Ohropax
高价位（500 元以上）：Bose Sleepbuds II、1More

### 五、总结建议
- 追求极致安静 + 预算有限 → 3M 泡沫耳塞
- 侧睡 + 舒适优先 → Mack's Pillow Soft 或 Loop Quiet
- 不想完全隔音 + 预算充足 → 电子睡眠耳塞
```

**Test A 特征：**
- 结构为自拟分类（类型/指标/推荐），无统一框架
- 无具体产品规格数据，无发布年份
- 价格为模糊区间，无来源
- 无市场趋势、无时机判断
- 不含任何引用来源

## 核心结论

> **无 Skill 的 AI 给的是"知道这件事存在"的印象式概述；**
> **有 Skill 引导的 AI 给的是"可以基于此做决策"的结构化专业交付物。**
>
> 本质差距：**认知框架的有无，决定了输出能否被直接使用。**
