# 05 创意写作测试

> [!info] 理论依据
> MT-Bench 的 Writing 类别，以及人类偏好评估中"创造性"维度。
> 核心问题：模型是否真有"创意"，还是只在模板里填词？

## 评估维度

| 维度 | 含义 |
|------|------|
| 原创性 | 避免陈词滥调，有新颖角度 |
| 结构感 | 开头-发展-结尾，节奏合理 |
| 语言质感 | 用词精准、有画面感 |
| 情感共鸣 | 读者能感受到情绪 |
| 任务完成度 | 是否满足了写作约束 |

---

## 测试题库

### 【诗歌】

**W1-01（约束创意）**
```
写一首关于"等待"的现代诗，要求：
- 不超过 12 行
- 不能出现"等待"、"思念"、"时间"三个词
- 至少用一个出人意料的意象
```

**W1-02（风格模仿）**
```
模仿余华的写作风格，用 100 字写一段关于"一个人在雨中等公交"的场景。
```

---

### 【故事开头】

**W2-01**
```
用以下开头续写一个故事（200字左右），要有反转：
"她第三次打开那扇门时，才意识到房间里的镜子从来就没有映出过她的脸。"
```

**W2-02（类型混搭）**
```
把"孙悟空大闹天宫"改写成一个现代职场故事，保留核心冲突，200字。
```

---

### 【实用创意写作】

**W3-01（文案）**
```
为一款"帮助焦虑症患者的冥想 App"写一句 Slogan 和一段 50 字的产品介绍，
风格：温柔、不夸张、有科技感。
```

**W3-02（标题党 vs 准确）**
```
给"研究发现每天步行 30 分钟可降低心血管风险 17%"写：
1. 一个标题党版本
2. 一个科学严谨版本
3. 一个适合朋友圈传播的版本
```

---

### 【创意质量鉴别题】

> [!tip] 用这道题区分"模板填充"和"真创意"

**W4-01**
```
写一段对话，两个椅子在讨论"被人坐"这件事，其中一个椅子是哲学家，
另一个是实用主义者。不超过 150 字。
```

---

## 评分 Rubric（满分 10）

| 分数段 | 标准 |
|--------|------|
| 9-10 | 有惊喜感，意象新颖，语言精准，满足所有约束 |
| 7-8 | 完成度高，有个别亮点，语言流畅 |
| 5-6 | 中规中矩，无明显错误，但缺乏特色 |
| 3-4 | 套用模板，陈词滥调，约束完成一半 |
| 1-2 | 流水账，或完全不符合要求 |

> [!warning] 扣分项
> - 用"在这片广袤的大地上"开头 -1
> - 诗歌每行都押同一个韵且内容无实质 -1
> - 明显 AI 味（"当然！很高兴帮你写..."）-2
