# 04 推理能力测试

> [!info] 理论依据
> - **GSM8K（Cobbe et al., 2021）**：小学数学应用题，考察多步推理
> - **ARC（Clark et al., 2018）**：科学推理选择题
> - **LogiQA**：逻辑推理（演绎/归纳/类比）
> - **Chain-of-Thought（Wei et al., 2022）**：推理过程可见性

## 测试子维度

```
推理能力
├── 数学推理（算术、代数、概率）
├── 逻辑推理（演绎、归纳、反证）
├── 常识推理（物理直觉、社会常识）
├── 空间推理（方位、图形变换）
└── 因果推理（原因→结果 / 反事实）
```

---

## 测试题库

### 【数学推理】（参考 GSM8K）

**R1-01（多步骤）**
```
小明以每小时 60 公里的速度从 A 城出发去 B 城，走了一半路程后，
因为路况变差改成 40 公里/小时。整个行程花了 2.5 小时，A、B 两城相距多少公里？
```
答案：120 公里（列方程：x/2÷60 + x/2÷40 = 2.5）

**R1-02（概率）**
```
一个袋子里有 3 个红球、4 个蓝球、3 个绿球。连续取两个球（不放回），
两个都是蓝球的概率是多少？
```
答案：4/10 × 3/9 = 12/90 = 2/15

**R1-03（陷阱题）**
```
一个农场有鸡和兔，共 20 个头，56 条腿，问各有多少只？
```
考察：经典题，看是否用方程组或暴力枚举，过程是否清晰

---

### 【逻辑推理】

**R2-01（演绎）**
```
所有的 A 都是 B，所有的 B 都是 C，小明是 A。
请问：小明是不是 C？小明是不是 B？能确定小明是 D 吗？
```

**R2-02（找矛盾）**
```
以下陈述中有没有逻辑矛盾？
"所有鸟都会飞。企鹅是鸟。企鹅在南极生活，它们在水中游泳但不能飞。"
```

**R2-03（反事实）**
```
如果爱因斯坦没有提出相对论，今天的 GPS 系统会有什么问题？
```
考察：因果链推理，GPS 依赖相对论修正原子钟

---

### 【常识推理】

**R3-01（物理直觉）**
```
一块冰放在水里，冰融化之后水面会上升、下降还是保持不变？为什么？
```
答案：阿基米德原理，水面基本不变（冰的质量=排开水的质量）

**R3-02（反直觉）**
```
你有一张 0.1mm 厚的纸，对折 42 次之后有多厚？直觉和计算差距大吗？
```
答案：2^42 × 0.1mm ≈ 439,804km，超过地月距离！

---

### 【思维链质量】

> [!tip] 关键测试点
> 不只看答案对不对，看**推理过程**是否清晰、步骤是否正确、中间步骤有无跳跃

**R4-01**
```
请一步步推导：1+2+3+...+100 等于多少？展示你的推理过程。
```
考察：是否使用高斯公式 n(n+1)/2，或逐步累加，过程可读性

## 评分标准

| 维度 | 权重 | 说明 |
|------|------|------|
| 答案正确性 | 40% | 最终答案 |
| 推理过程清晰度 | 35% | 步骤可读，无跳跃 |
| 错误识别 | 15% | 发现陷阱/矛盾 |
| 表达简洁性 | 10% | 不啰嗦，不过度铺垫 |
