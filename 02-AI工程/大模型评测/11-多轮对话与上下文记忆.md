# 11 多轮对话与上下文记忆

> [!info] 和 [[02-对话连贯性测试]] 的区别
> - 02 测的是：**短对话中**是否引用了上文（3-5轮）
> - 本文测的是：**跨轮次的记忆质量**，包括长对话记忆衰减、跨话题记忆召回、偏好记忆

> [!info] 理论依据
> - **LongMemEval（Wu et al., 2024）**：长对话记忆召回基准，5大记忆类型
> - **MemGPT（Packer et al., 2023）**：分层记忆管理架构
> - **MSC（Multi-Session Chat，Xu et al., 2022）**：跨会话记忆保留测试

## 记忆类型分类（LongMemEval）

```
记忆类型
├── 事实记忆：用户说过的客观信息（姓名、职业、地点）
├── 偏好记忆：用户表达的喜好（不吃辣、喜欢简洁风格）
├── 指令记忆：用户设定的规则（"回答不超过100字"）
├── 事件记忆：发生过的事（"上次我提到我要去北京出差"）
└── 否定记忆：用户明确说"不"的（"不要再建议我用Excel"）
```

## A. 单会话长对话记忆测试

### 【记忆注入 + 延迟召回】

**对话设计：先埋信息，几轮后考察**

```
轮1：用户说："我叫李明，在杭州做电商，主要卖宠物用品"
     → 正常对话（帮忙写文案等）

轮2-5：正常其他话题

轮6：用户问："帮我推荐一个适合我的城市的 SEO 工具"
     → 考察：是否记住"杭州"这个城市

轮7：用户问："我们公司主要卖什么来着，我想写个简介"
     → 考察：是否记住"宠物用品"，而非让用户重复
```

**M1-01（事实记忆）**
```
轮1：我是一名高中数学老师，学生大概 17 岁左右
轮2：帮我解释勾股定理
轮3：用更简单的方式
轮4：给我出一道练习题
--- 5轮其他话题 ---
轮9：帮我写一道关于三角函数的练习题
```
考察轮9：是否自动适配"高中难度"，而非出大学题

---

### 【偏好记忆与持续遵守】

**M2-01**
```
轮1：用户说："我不喜欢废话，每次回答不超过3句话"
轮2-8：各种问题
轮9：复杂问题（比如解释量子力学）
```
考察：轮9是否还在遵守"3句话"的约束，还是因为问题复杂就"忘了"

**M2-02（否定记忆）**
```
轮1：用户说："以后不要用列表，我更喜欢段落式回答"
轮3：一个适合用列表的问题
轮6：另一个适合用列表的问题
```
考察：是否持续避免列表，还是"记了两轮就忘了"

---

### 【记忆冲突处理】

**M3-01**
```
轮1：用户说"我是素食主义者"，推荐食谱
轮5：用户说"帮我推荐一个烤肉餐厅"
```
考察：如何处理新指令与已知偏好的冲突？
好的模型：友好确认"您之前提到是素食，请问是特殊场合吗？"
不好的模型：直接推荐烤肉（忘记了）或拒绝（过于死板）

**M3-02（更新记忆）**
```
轮1：用户说"我住在北京"
轮4：用户说"我刚搬到上海"
轮7：用户问"附近有什么好玩的"
```
考察：用"上海"而非"北京"，能更新而非叠加

## B. 跨话题记忆召回

**对话组 Cross-1**
```
话题A（轮1-4）：讨论 Python 编程
话题B（轮5-8）：讨论旅行计划去日本
话题C（轮9-11）：讨论健身饮食

轮12：用户说"结合我之前说的旅行，帮我想一个在日本也能坚持的健身方案"
```
考察：跨话题（B+C）信息整合召回

## C. 记忆衰减测试

> 测试在长对话中，早期信息的保留率

```
第1轮注入信息：
- 用户名：陈浩
- 预算：5000元
- 目标：给女朋友买礼物，她喜欢简约风
- 限制：不要买化妆品

第10轮测试（中间穿插9轮无关对话）：
"帮我总结一下你对我需求的理解"

第20轮测试（中间又10轮无关对话）：
"现在给我推荐3个具体商品"
```

**评分：记住了几个信息点**

| 信息点 | 第10轮还记得 | 第20轮还记得 |
|--------|------------|------------|
| 用户名陈浩 | | |
| 预算5000 | | |
| 给女朋友 | | |
| 简约风 | | |
| 不要化妆品 | | |
| 总分（/5） | | |

## D. 跨会话记忆（需持久化记忆功能）

> 适用于有记忆插件/持久化功能的模型（如 MemGPT、OpenClaw 等）

**Session-1**
```
用户设定：我叫王芳，32岁，做 UI 设计，不喜欢冗长回答
[正常对话若干轮]
```

**Session-2（新会话）**
```
用户说："继续上次的设计讨论"
或直接问："你还记得我是做什么的吗？"
```

评分：
- 5分：完整记住所有关键信息
- 3分：记住部分（姓名/职业但忘了偏好）
- 0分：完全不记得（全新对话）

---

## 评分标准

| 记忆类型 | 权重 | 衡量方式 |
|----------|------|---------|
| 事实记忆召回 | 30% | 信息点复现率 |
| 偏好持续遵守 | 30% | 整个对话期间遵守率 |
| 记忆冲突处理 | 20% | 是否优雅处理矛盾 |
| 长轮次衰减率 | 20% | 20轮后仍保留的信息% |
