## 安装 Docker 步骤（WSL2 Ubuntu）

在 WSL2 终端中执行：

```bash
# 1. 安装 Docker Engine（官方源）
curl -fsSL https://get.docker.com | sh

# 2. 将当前用户加入 docker 组（免 sudo）
sudo usermod -aG docker $USER

# 3. 启动 Docker 服务
sudo service docker start

# 4. 验证安装
docker --version
docker compose version

# 5. 设置开机自启（可选，WSL2 需要 systemd 支持）
sudo systemctl enable docker
```

> [!note] 替代方案
> 也可以安装 [Docker Desktop for Windows](https://www.docker.com/products/docker-desktop/)，启用 WSL2 后端集成，Docker 会自动在 WSL2 发行版中可用。

## 结论

- WSL2 环境正常，资源充足（16 核 / 7.7 GiB 内存 / 942 GiB 磁盘）
- **需要安装 Docker** 后才能进行 go2rtc 部署
- 建议在 WSL2 内直接安装 Docker Engine（更轻量），或使用 Docker Desktop（更方便）
