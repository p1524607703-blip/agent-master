# 07 中文能力专项测试

> [!info] 理论依据
> - **C-Eval（Huang et al., 2023）**：52个学科的中文知识评测
> - **CMMLU（Li et al., 2023）**：中文多任务语言理解
> - **CLUE（Xu et al., 2020）**：中文语言理解评测基准

## 为什么单独测中文？

多数模型英文能力 > 中文能力，C 端中文用户需要额外关注：
- 中文语义理解（多义词、歇后语、隐喻）
- 中国文化知识（历史、习俗、当代语境）
- 中文写作质量（不是英文翻译腔）
- 方言/网络用语理解

---

## 测试题库

### 【语义理解：多义与歧义】

**C1-01**
```
"这道题我做了两个小时还没做出来"和
"这锅菜我做了两个小时终于做好了"，
两句话中"做"的意思有什么不同？
```

**C1-02**
```
"他这个人很有意思"和"这件事很有意思"，
两个"有意思"的含义是否相同？
```

**C1-03（歇后语）**
```
"猪八戒照镜子——里外不是人"这句歇后语是什么意思？
在什么情境下会用到？
```

---

### 【文化知识】

**C2-01（中国历史）**
```
"靖康之耻"发生在哪个朝代？为什么这个事件对中国历史影响深远？
```

**C2-02（当代文化）**
```
解释以下网络用语的含义和使用场景：
"yyds"、"内卷"、"躺平"、"破防"
```

**C2-03（习俗礼仪）**
```
在中国，红包的金额为什么要避开"4"？
葬礼和婚礼场合各有哪些常见禁忌？
```

---

### 【中文写作质量】

**C3-01（无翻译腔）**
```
把这段英文翻译成自然的中文，不要有翻译腔：
"The elephant in the room is that no one wants to talk about
how the new policy will affect small businesses."
```
考察：能否理解"elephant in the room"并用地道中文表达（比如"显而易见却无人提及的问题"）

**C3-02（地道表达）**
```
帮我把这句话改得更口语化、更有中文味：
"我对于这个项目的推进进度表示担忧，认为应当采取更为积极主动的措施。"
```

**C3-03（文言理解）**
```
翻译并解释这段文言文：
"学而不思则罔，思而不学则殆。"
联系现代生活举一个例子。
```

---

### 【中文推理】

**C4-01（中文逻辑题）**
```
有甲、乙、丙三人，一人是老师，一人是医生，一人是律师。
已知：甲不是老师，乙不是医生，丙不是律师，乙也不是老师。
请问三人各是什么职业？
```
答案：甲=医生，乙=律师，丙=老师

**C4-02（成语运用）**
```
"南辕北辙"和"缘木求鱼"有什么共同点？
分别造一个句子，体现它们的细微差别。
```

---

### 【方言与俚语（理解层面）】

**C5-01**
```
"整个人都不好了"和"整个人都好了"，
语气和含义上有什么反常规的差别？
```
考察：理解反语/夸张的网络语义

**C5-02**
```
广东人说"你饮咗未"是什么意思？
这体现了粤语饮食文化的什么特点？
```
