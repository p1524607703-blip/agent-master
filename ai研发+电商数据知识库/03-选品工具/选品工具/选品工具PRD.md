## 一、产品定位

**产品名称**：暂定（待命名）
**产品类型**：跨境电商选品 SaaS 工具（Web 端）
**核心理念**：比卖家精灵更轻量、更易用，让新手也能看懂数据，做出选品决策

---

## 二、目标用户

| 用户类型 | 优先级 | 描述 |
|---------|------|------|
| 跨境电商新手 | 🥇 P0 | 不懂数据分析，需要傻瓜式结论和引导 |
| 有经验的卖家 | 🥈 P1 | 需要多维度数据支撑决策 |
| B 端企业/团队 | 🥈 P1 | 需要团队协作、批量分析、定时监控 |

> [!tip] 设计原则
> 新手能用，专业够用。界面默认新手友好，专业模式按需切换。

---

## 三、核心功能

### 3.1 数据来源（按优先级）

| 阶段 | 平台 | 数据类型 | 更新频率 |
|------|------|---------|---------|
| MVP | Reddit | 热帖讨论、产品提及 | 每天 |
| MVP | Google Trends | 搜索趋势、热度曲线 | 每天 |
| P1 | Facebook 广告库 | 在投广告、爆款素材 | 每小时 |
| P2 | TikTok | 热门视频、爆款商品 | 每天 |
| 暂缓 | 小红书 | 种草内容 | 待定 |

### 3.2 核心模块

**① 趋势发现**
- 展示各平台近期上升趋势商品
- 热度评分 + 趋势折线图
- 支持关键词搜索和类目筛选

**② 竞争评估**
- 市场竞争热度
- 在售商品数量、价格区间
- 利润空间估算

**③ 竞品广告监控**
- Facebook 广告库数据
- 竞品投放时长、创意类型
- 关键词广告分布

**④ AI 选品分析**（接入 OpenClaw）
- 新手模式：一句话结论 + 理由（通俗语言）
- 专业模式：多维评分 + 结构化报告
- 用户可自由切换

**⑤ 选品看板**
- 个人/团队收藏夹
- 自定义筛选条件
- 数据导出

### 3.3 AI Agent 自动监控

> [!note] OpenClaw 集成
> 支持 OpenClaw AI Agent 接入，agent 每天定时访问平台执行监控任务，分析数据后推送日报。

- **触发方式**：定时任务（每日）
- **分析引擎**：OpenClaw agent 调用站内数据 API
- **推送渠道**：微信 / 飞书 / 邮件（用户自选）
- **报告内容**：今日爆款、趋势变化、值得关注的选品机会

---

## 四、界面设计原则

- **轻量化**：首屏信息不超载，核心指标突出展示
- **可视化优先**：趋势图、热力图、评分卡片替代纯数字表格
- **双模式**：新手模式（大字、结论前置）/ 专业模式（完整数据）
- **参考对标**：比卖家精灵更简洁，接近 Exploding Topics 的克制风格
