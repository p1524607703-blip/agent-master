## 一、设计理念

### 1.1 TikTok Web 风格核心特征

基于 TikTok Web 端（`tiktok.com`）的视觉语言：

| 维度 | TikTok Web 风格 | 传统管理后台风格 |
|------|----------------|----------------|
| 背景色 | `#010101` 纯黑 + `#121212` 卡片 | `#F5F5F5` 浅灰 + 白色卡片 |
| 主色调 | `#FE2C55`（TikTok 红）+ `#25F4EE`（TikTok 青） | 蓝色/紫色品牌色 |
| 文字色 | `#FFFFFF`（主）+ `#86878B`（次） | `#333333` + `#666666` |
| 圆角 | 小圆角 4-6px，TikTok 偏好锐利感 | 大圆角 8-16px |
| 间距 | 紧凑 8-12px，更像移动端 | 宽松 16-24px |
| 卡片 | 深色卡片 `#1F1F1F`，无描边纯色 | 白色卡片 + 阴影或描边 |
| 滚动 | 无限滚动，短视频卡片网格 | 分页表格 |
| 动画 | 微交互丰富，hover scale/shadow 明显 | 简单过渡 |

## 二、设计规范（TikTok 风格）

### 2.1 色彩系统

```css
/* TikTok 品牌色 */
--tiktok-pink:     #FE2C55;   /* TikTok 主色/CTA */
--tiktok-cyan:      #25F4EE;   /* TikTok 强调色 */
--tiktok-dark:      #010101;   /* 页面背景 */
--tiktok-darker:    #0D0D0D;  /* 次级背景 */
--tiktok-card:      #1F1F1F;  /* 卡片背景 */
--tiktok-card-hover: #2A2A2A; /* 卡片悬停 */

/* 文字层级 */
--text-primary:      #FFFFFF;
--text-secondary:    #86878B;
--text-disabled:     #545454;

/* 功能色 */
--color-success:     #00D4AA;
--color-warning:     #FFD60A;
--color-error:       #FF3B5C;
--color-info:        #4FC3F7;

/* 状态色 */
--status-fire:       #FE2C55;  /* 爆款标记 */
--status-cold:       #545454; /* 一般标记 */

/* 渐变 */
--gradient-tiktok:   linear-gradient(135deg, #FE2C55 0%, #25F4EE 100%);
--gradient-pink:     linear-gradient(135deg, #FE2C55, #FF6B8A);
```

### 2.2 字体规范

```css
/* 字体栈 */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Noto Sans SC', sans-serif;

/* 字号层级 - 更紧凑 */
--text-xs:    11px;   /* 微注释/时间戳 */
--text-sm:    12px;   /* 标签/次要信息 */
--text-base:  13px;   /* 正文/默认 */
--text-lg:    14px;   /* 卡片标题/按钮 */
--text-xl:    16px;   /* 页面标题 */
--text-2xl:   18px;   /* 大标题 */
--text-3xl:   20px;   /* Hero 标题 */

/* 字重 */
font-weight: 400;   /* 正文 */
font-weight: 500;   /* 中等 */
font-weight: 600;   /* 标题/按钮 */
font-weight: 700;   /* 强调 */

/* 行高 - 紧凑 */
line-height: 1.3;   /* 标题 */
line-height: 1.5;   /* 正文 */
```

### 2.3 间距系统（更紧凑）

```css
--space-1:   4px;    /* 微分隔 */
--space-2:   8px;    /* 组件内小间距 */
--space-3:   12px;   /* 表单项/卡片内 */
--space-4:   16px;   /* 区块间距 */
--space-5:   20px;   /* 容器间距 */
--space-6:   24px;   /* 大区块 */
--space-8:   32px;   /* 页面边距 */

/* 圆角 - TikTok 偏好小圆角 */
--radius-sm:  4px;
--radius-md:  6px;
--radius-lg:  8px;
--radius-xl:  12px;
--radius-full: 999px;  /* Pill 按钮 */
```


### 6.3 API 接口（视频详情页）

```typescript
// 获取视频详情
GET /api/video/:id

// 视频 AI 分析
POST /api/video/:id/analyze
Response: {
  "frames": FrameAnalysis[],
  "evaluation": VideoEvaluation,
  "category": VideoCategory,
  "virality_analysis": ViralityAnalysis,
  "copy_analysis": CopyAnalysis,
  "engagement": EngagementMetrics,
  "improvements": Improvement[]
}

// 优化改写
POST /api/video/:id/optimize
Request: { copy_rewrite_level: 'light' | 'medium' | 'aggressive' }
Response: {
  "optimized_copy": OptimizedCopy,
  "visual_script": VisualScript
}

// 下载无水印视频
GET /api/video/:id/download?format=mp4

// 下载音频
GET /api/video/:id/audio

// 导出分析报告
POST /api/video/:id/export
Request: { format: 'pdf' | 'csv' | 'markdown' }
Response: { download_url: string }
```
